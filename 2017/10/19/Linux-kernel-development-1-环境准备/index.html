其他一些类似命令，这里仅列出：

  1. make oldconfig 使用已经存在的 ./.config的内容，将里面的原本值作为新config的设定值。新config中涉及的新特性则列出来让用户选择。
  2. make gconfig 用Gtx图形界面来配置。需要X window支持。
  3. make xconfig 用Qt图形界面来配置。需要X window支持。
  4. make config
