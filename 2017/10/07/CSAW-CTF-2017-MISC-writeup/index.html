初次连上后，会给出一个字符串 00110010001，根据偶校验，选择发送是0还是1，然后服务器端会再返回下一个字符串。

从给出的提示信息，以及结合串口通信标准来看，每次我们收到的字符串总共是11位，其中第一位都是0，表示数据的开始，剩下的数据满足8-1-1，是指有8位数据位，1位校检位，以及1位的停止位。

比如说：00110010001

起始位 数据位 奇偶校验位 停止位
0 01100100 0 1
